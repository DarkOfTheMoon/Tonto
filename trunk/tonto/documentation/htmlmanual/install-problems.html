Known problems installing TONTO

Compiler specific issues:

  • Digital Fortran 5.0 for OSF/1 run time libraries 376 or later must be used. Before this, the for_cpusec routine will be unresolved. Also, in the site_config file you must have -DLANGUAGE_FORTRAN_90 on the DEFS line.
