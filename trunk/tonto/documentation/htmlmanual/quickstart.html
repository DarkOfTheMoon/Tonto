Building and Installing Tonto

The Quick Way

Download the latest version of TONTO from the download page.

To unpack the archive you downloaded, type: 

gunzip -c tonto.tar.gz | tar xvf -

Then to compile and install Tonto, type: 

cd tonto
./configure
make install

"configure" may tell you to edit a specific file in the site_config directory which contains default settings.
If this is the case, it is because "configure" does not know which options to set for the compiler on your system. See the site_config documentation.

You may want to have a look at the site_config file that "configure" tells you anyway, just to check the compiler settings.

Some compilers may need upgrading or require special settings. Have a look at the compiler specific issues page for these.
