Contributing code to TONTO

Remember, TONTO is copyleft under the less restrictive GNU library license. It is free software.

If you want features added or corrected in TONTO, you can do it yourself and mail the new stuff to us.

If you intend to make major and continual additions, it may be better to contact us: we will give you access to the current CVS repository so you can join the team working on the code.

Either way, your code contributions will be very much appreciated.

Your rights as a contributing author are protected under the GNU license.

You can find out more of the underlying philosophy and aspirations of the TONTO project in the final chapter of this manual.
