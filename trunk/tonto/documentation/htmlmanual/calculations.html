How do I use TONTO for calculations?

The executables for TONTO may require a text input file. The structure of the input files follows a well defined database protocol. Input files are discussed in Chapter 3, Using TONTO to do calculations, and example inputs are given in chapter Chapter 4, Examples of TONTO input files. The actual keywords are documented in Chapter 5, Keyword documentation for TONTO including a description of how to use the online documentation to find keywords.

To make good use of TONTO, especially the online WWW documentation, it will help if you understand a little about how the overall program is structured. This is explained in the section called “The design of TONTO”.
