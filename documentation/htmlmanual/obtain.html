How do I get TONTO?

The web site is www.theochem.uwa.edu.au/tonto/.

The web site contains the most up to date version of TONTO and this manual, and the full online documentation of the program.

What do I need to install and use TONTO?

TONTO is a Unix based program. You will need some Unix knowledge, including how to download files using your browser, how to unzip files, how to list and edit files, and how to run executables.

To download and compile TONTO, you will need:

  • a Fortran95 compiler.
  • the GNU utilites gunzip and gmake.
  • the perl program.
  • the programs tar and cmp, available in most operating systems.
