System requirements for TONTO

If you are not doing development work and have a TONTO executable, then you do not need any of the following programs.

Apart from a Fortran compiler, you will need the following free software:

  • Mozilla or another web browser

  • tar or winzip for extracting the distribution

  • gunzip or bunzip2 for uncompressing source files

  • gmake for controlling the compilation

  • perl for the foo preprocessor

  • CVS (only for developers)

  • The BLAS and LAPACK numerical libraries for linear algebra (only if not available on your system)

The above software must be installed and working before you start compilation.
