How is TONTO different from other packages?

TONTO is not just another computational chemistry package; it also provides an environment that makes code development a simple task. The modular form of TONTO makes its use logical and its input simple.

TONTO has been implemented in foo, an object oriented preprocessor which currently translates into Fortran951. Foo makes the process of maintaining and documenting the TONTO system automatic. In particular, the online html documentation is generated directly from the code.

TONTO can perform many tasks in quantum crystallography. The aim of this field is to derive accurate molecular wavefunctions by incorporating x-ray, polarised neutron, or other diffraction and scattering data directly into the method for determining the wavefunction.

There is no guarantee that TONTO will remain translated into Fortran95. A translation into C++, or more likely another object oriented language, would also be relatively easy and may be required at a later date in the event that Fortran ceases to be the best option for fast code. Thus, the lifespan of code written in TONTO can be expected to be quite long.

The best feature of TONTO is that it is free. The modules of TONTO can be used without modification in your own work, commercial or otherwise2.

Footnotes
