TONTO Installation Introduction

If your Fortran compiler and operating system is known to us, you are in luck. TONTO has been compiled and tested on it, and the installation is very easy.

The set of known systems is given in the next section.

In addition, you will need a number of free software tools, which are also listed in the next section.

If your system is not known to us, you can look at the section entitled Problems installing TONTO. Except for compiler bugs, it should still be straightforward to get TONTO to work. We are always interested in porting our code to your system; please contact us.

If you own a Silicon Graphics machine, TONTO will not work. At the current time, the Silicon Graphics Fortran95 compiler cannot be made to work for the style of programming that we use.

Requirements for compiling TONTO

You will need a Fortran 95 conforming compiler.
The set of compilers and operating systems on which TONTO has been tested is:

  • Fujitsu frt compiler under UXP5
  • Fujitsu frt compiler under LINUX
  • NAG f95 compiler under LINUX
  • IBM xlf compiler under AIX
  • DEC f95 compiler under OSF1
  • Compaq f95 compiler under TRU64 unix
  • Sun frt compiler

Apart from a Fortran compiler, you will need the following free software:

  • Netscape or another web browser to view the documentation.
  • the GNU utilites gunzip and gmake.
  • the perl program.
  • the programs tar and cmp, available in most operating systems.
